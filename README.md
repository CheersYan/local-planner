Local Planner (scaffold)
========================

Local-only planning app scaffolded with Next.js, TypeScript, Tailwind CSS, and ESLint. The home page currently contains a single placeholder heading.

Getting Started
---------------

Install dependencies (pnpm only):

```
pnpm install
```

Run the dev server:

```
pnpm dev
```

Open http://localhost:3000 to view the placeholder page. Edit `src/app/page.tsx` to continue building.
